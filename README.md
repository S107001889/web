# OWASP

##  A1:2017-Injection
  漏洞說明：
  1.SQL注入(SQL Injection, SQLi)
    SQL注入可以用來入侵使用資料庫的軟體，被開放網路軟體安全計畫（OWASP）視為侵入網站及SQL資料庫最常見的技術之一，SQLi將惡意SQL語法插入輸入字串(作為 
    指令或查詢的一部分)，並且利用了軟體或網頁應用程式的漏洞。
    -例如沒有正確地過濾使用者輸入。比方說，攻擊者可以將惡意輸入變成SQL查詢的參數，讓資料庫認為是SQL指令的一部分而執行。
